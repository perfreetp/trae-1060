## 1. 产品概述

流域水库群调度 Web 应用是面向防汛调度人员的专业决策支持平台，在汛期提供雨情、水情、库容变化的统一监控与调度管理能力。通过信息化手段提升水库群联合调度效率，降低洪涝灾害风险。

- 核心目标：实现流域内多水库的联合调度智能化、决策可视化、指令传递高效化
- 目标用户：防汛指挥中心调度人员、水库管理人员、值班人员

## 2. 核心功能

### 2.1 用户角色

| 角色 | 描述 | 核心权限 |
|------|------|----------|
| 调度员 | 防汛指挥中心调度人员 | 查看总览、制定方案、下达指令、执行回填 |
| 值班员 | 现场值班人员 | 查看水情、执行指令、值班交接 |
| 管理员 | 系统管理员 | 用户管理、系统配置、日志审计 |

### 2.2 功能模块

1. **流域总览**：流域地图、关键指标卡片、实时预警、态势总览
2. **雨量站**：雨量站点分布、实时降雨量、过程线对比、阈值预警
3. **河道水位**：河道站点分布、实时水位、历史趋势、超警标注
4. **单库详情**：水库基本信息、库容曲线、泄流能力查询、来水预报录入
5. **联合调度方案**：方案模拟、风险点标注、多方案对比、方案审批
6. **调度指令**：指令下达、执行状态跟踪、短信提醒、执行回填
7. **复盘评估**：历史洪水检索、调度日志、复盘评分、问题清单管理

## 3. 核心流程

### 3.1 调度决策主流程

```mermaid
flowchart LR
    A["雨情水情监测"] --> B["阈值预警触发"]
    B --> C["查看单库/河道详情"]
    C --> D["录入来水预报"]
    D --> E["制定联合调度方案"]
    E --> F["方案模拟验证"]
    F --> G["方案审批"]
    G --> H["下达调度指令"]
    H --> I["短信通知执行人"]
    I --> J["执行回填反馈"]
    J --> K["调度过程记录"]
    K --> L["事后复盘评估"]
```

### 3.2 值班交接流程

```mermaid
flowchart LR
    A["接班人员登录"] --> B["查看当前水情雨情"]
    B --> C["查看待办事项"]
    C --> D["确认交接内容"]
    D --> E["填写接班记录"]
    E --> F["完成交接"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝色系（#0A2463 主色、#3E92CC 辅助色），体现专业、稳重、科技感
- **预警色**：红色（#D62828 严重）、橙色（#F77F00 较重）、黄色（#FCBF49 一般）、蓝色（#0096C7 提示）
- **按钮风格**：扁平化设计，圆角 4px，悬停有微光影效果
- **字体**：中文使用 "Noto Sans SC"，数字使用 "Roboto Mono" 等宽字体
- **布局风格**：顶部导航 + 侧边菜单 + 内容区三栏布局，数据以卡片 + 图表形式呈现
- **图标风格**：使用 Lucide 线性图标，简洁专业

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 流域总览 | 地图区域 | 深色底图 + 发光节点 + 连接线，hover 显示详情 |
| 流域总览 | 指标卡片 | 白色卡片 + 数据大屏风格数字 + 趋势小箭头 |
| 流域总览 | 预警面板 | 列表式，按严重程度排序，红色高亮闪烁 |
| 雨量站 | 站点表格 | 斑马纹表格，超阈值行标红 |
| 雨量站 | 过程线图 | ECharts 折线图，多色区分不同站点 |
| 单库详情 | 库容曲线 | 双 Y 轴图表，水位/库容同步显示 |
| 单库详情 | 泄流查询 | 表单 + 结果表格，支持参数调整 |
| 联合调度方案 | 模拟面板 | 滑块调整泄流量，图表实时更新模拟结果 |
| 调度指令 | 指令列表 | 时间线布局，状态标签色彩区分 |
| 复盘评估 | 评分面板 | 雷达图展示多维度评分，进度条显示得分 |

### 4.3 响应式

- 桌面端优先，最小支持宽度 1366px
- 关键数据区域支持自适应缩放
- 表格支持横向滚动
- 移动端做基础适配，保证核心功能可用

### 4.4 动效设计

- 页面切换：淡入淡出过渡
- 数据更新：数字滚动动画
- 预警触发：呼吸灯闪烁效果
- 卡片悬停：上移 + 阴影加深
- 图表加载：渐进式绘制动画
